Formatações de Texto

Negrito / Itálico

Utilizamos a tag <STRONG> para NEGRITO e a tag <EM> para ITÁLICO.

Texto Marcado / Texto Grande e Pequeno

Utilizamos a tag <MARK> para MARCAR. Utilizamos a tag <BIG> para DEIXAR TEXTO GRANDE e a tag <SMALL> para DEIXAR TEXTO PEQUENO.

Texto Deletado / Texto Inserido

Utilizamos a tag <DEL> para TEXTO DELETADO e a tag <INS> para TEXTO INSERIDO.

Texto Sobrescrito / Texto Subscrito

Utilizamos a tag <SUP> para TEXTO SOBRESCRITO e a tag <SUB> para TEXTO SUBSCRITO.

Formatações de Texto Extras

Código-Fonte

Utilizamos a tag <CODE> para códigos: document.getElementById("teste")

Utilizamos a tag <PRE> para códigos com formatação preservada:

        
      num = int(input("Digite um número: "))
      if num % 2 == 0:
          print("Número par")
      else:
          print("Número ímpar")
      print("Fim do programa")

Citação / Citação Completa

Utilizamos a tag <Q> para citações: O Computador é um burro muito rápido

Utilizamos a tag <BLOCKQUOTE> para citações completas:

A Diferença entre um programador e um não programador é que o programador sabe como fazer o computador trabalhar para ele, enquanto o não programador trabalha para o computador.

Abreviações

Utilizamos a tag <ABBR> para abreviações: HTML, CSS e JS.
