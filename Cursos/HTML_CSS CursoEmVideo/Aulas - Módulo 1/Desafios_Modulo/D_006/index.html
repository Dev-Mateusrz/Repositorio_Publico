Tags HTML5

HTML5 introduziu várias tags novas para melhorar a semântica e a estrutura dos documentos web. Algumas das principais tags incluem:

  • <header>: Define o cabeçalho de uma seção ou página.
  • <nav>: Define uma seção de navegação.
  • <section>: Define uma seção genérica de um documento.
  • <article>: Define um conteúdo independente, como um artigo de blog.
  • <aside>: Define conteúdo relacionado, como barras laterais.
  • <footer>: Define o rodapé de uma seção ou página.
  • <figure> e <figcaption>: Usadas para agrupar mídia e sua legenda.
Essas tags ajudam a tornar o código mais legível e melhoram a acessibilidade e SEO dos sites.
